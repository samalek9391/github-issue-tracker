1️⃣ What is the difference between var, let, and const?
=> The difference between var, let, and const are Var only maintains function scope but not global and block scope, but let and const maintain all scopes.

2️⃣ What is the spread operator (...)?
=> In JavaScript, the spread operator (...) is used to spread (expand) the values ​​inside an iterable (such as an array, string) or an object.

3️⃣ What is the difference between map(), filter(), and forEach()?
=>map(), filter(), and forEach() are array methods in JavaScript. map() creates a new array by changing each element; filter() creates a new array with elements that match a condition and forEach() → Loops through the array but does not return a new array.

4️⃣ What is an arrow function?
=> Arrow Function is a small and modern way to write functions in JavaScript.
It is introduced in ES6 (ECMAScript 2015).

Arrow function is written using => symbol.
This => is called arrow.

5️⃣ What are template literals?
=> Template literals are a way to write strings in JavaScript using backticks ( ) instead of quotes. It allows to insert dynamically using ${}.
